# 学童シフト＆人件費管理システム

学童保育のスタッフ シフト管理と人件費シミュレーションを一元化する Web アプリケーションです。

## 🚀 クイックスタート

```bash
# 1. 依存パッケージをインストール
npm install

# 2. サーバーを起動
npm start

# 3. ブラウザでアクセス
# http://localhost:3000
```

## 📋 主な機能

| 機能 | 説明 |
|------|------|
| **スタッフマスタ** | 名前・時給/月給を登録・管理 |
| **シフト希望収集** | スタッフがスマホからシフト希望を入力 |
| **シフト表** | 月間カレンダー形式で出退勤時間を管理 |
| **希望→確定 自動転記** | ワンクリックでシフト希望を確定シフトに反映 |
| **人件費シミュレーション** | 時給/月給に応じた人件費を自動計算 |
| **CSVエクスポート** | シフト・人件費データをCSVで出力 |

## 📱 使い方

### 管理者
1. `http://localhost:3000` にアクセス
2. 組織を作成（組織コードが自動発行）
3. スタッフを登録（名前・時給/月給）
4. シフト表で出退勤時間を管理
5. 人件費シミュレーションで概算を確認

### スタッフ
1. 管理者から共有された URL（`http://サーバー:3000/staff/組織コード`）にアクセス
2. 自分の名前を選択
3. 出勤可能な日をタップして時間を入力
4. 「送信」ボタンで希望を提出

## 🏢 他社への共有

このシステムはマルチテナント対応です。組織ごとに固有のコードが発行されるため、1つのサーバーで複数の学童が利用できます。

### サーバー設置方法
```bash
# サーバーマシンで
git clone <このリポジトリ>
cd gakudo-shift-manager
npm install
npm start
```

### ポート変更
```bash
PORT=8080 npm start
```

## 🗄 データ

- データは `data/gakudo-shift.db`（SQLite）に保存されます
- バックアップは このファイルをコピーするだけでOK
- CSV エクスポートでデータを外部保存することも可能
